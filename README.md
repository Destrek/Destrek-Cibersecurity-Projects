# Destrek Cibersecurity Projects

Este repositorio reúne en un único espacio todas las herramientas, proyectos y experimentos que vaya desarrollando en el ámbito de la Ciberseguridad. Aquí compartiré:

-  Scripts en Python y Bash.
-  Honeypots.
-  Herramientas ofensivas y defensivas.
-  Write-ups de máquinas y laboratorios.
-  Experimentos y proyectos personales relacionados con la propia Ciberseguridad.

Objetivo del repositorio:

- Centralizar mis proyectos de Ciberseguridad en un único espacio.
- Mantener una estructura clara y escalable.
- Documentar cada herramienta o proyecto para compartirla y hacerla reproducible y que pueda ser de utilidad para otros.
- Compartir conocimiento y mejorar mi aprendizaje así como ayudar al aprendizaje de otros usuarios interesados en este campo.


Estado del repositorio:

Este repositorio está en construcción y evolucionará con el tiempo.
A medida que vaya creando herramientas, subiendo write-ups de máquinas y haciendo ciertos laboratorios iré organizando el propio repositorio en carpetas específicas.
